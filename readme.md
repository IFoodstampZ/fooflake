"An example system flake for learning how to use Nix, Nix-Darwin, and NixOS!"
This flake is meant to be as simple as possible, providing a sensible foundation for learning how to do things the Nix way.

included tutorials:
* nixos
* nix-darwin
* home-manager
* wrappers
* mnw - a minimal neovim wrapper (gergware)
